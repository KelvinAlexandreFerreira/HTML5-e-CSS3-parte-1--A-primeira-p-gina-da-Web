Hierarquia no CSS
Peso de cada hierarquia do CSS:
tag    -> 1
class  -> 10
id     -> 100
inline -> 1000

Os valores de peso podem ser somados, ex:
p       -> 1
form p  -> 2
.teste  -> 10
p.teste -> 11
#teste  -> 100
